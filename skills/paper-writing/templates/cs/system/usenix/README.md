# USENIX Conference Skeleton

## Format Constraints
- **Class:** `article` with `usenix2019_v3.sty`
- **Page limit:** depends on venue (typically 12 pages submission / 14 pages camera-ready for major conferences; 8 pages for workshops like HotStorage)
- **Column:** two-column (`letterpaper,twocolumn,10pt`)
- **References:** unlimited, do not count toward page limit
- **Style:** `\bibliographystyle{plain}`
- **Artifact evaluation:** `usenixbadges.sty` included for optional artifact badges

## Typical Venues
OSDI, FAST, NSDI, ATC, HotStorage, HotOS — USENIX Association conferences

## Page Budget
See the matching blueprint in `templates/cs/system/blueprints/`:
- 12 pages: [blueprints/12pages.md](../blueprints/12pages.md) — OSDI, FAST, NSDI, ATC
- 8 pages: [blueprints/8pages.md](../blueprints/8pages.md) — HotStorage, HotOS

## Required Files
This skeleton contains `usenix2019_v3.sty`, `usenixbadges.sty` (+ badge PDFs), a `main.tex` preamble, `references.bib` placeholder, `appendix.tex` and `artifacts.tex` stubs. Copy to `paper/` to begin writing.
