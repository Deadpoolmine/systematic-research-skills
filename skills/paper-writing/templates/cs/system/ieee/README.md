# IEEE Conference Skeleton

## Format Constraints
- **Class:** `IEEEtran.cls` (IEEE conference format)
- **Page limit:** varies by venue (typically 8-10 pages)
- **Column:** two-column
- **References:** unlimited, do not count toward page limit
- **Style:** `\bibliographystyle{IEEEtran}`

## Typical Venues
MSST, INFOCOM, IPDPS, and other IEEE computer society conferences

## General Chapter Planning
IEEE papers typically follow:
1. Introduction
2. Background & Motivation
3. Design / Architecture
4. Implementation
5. Evaluation
6. Related Work
7. Discussion / Limitations
8. Conclusion

## Required Files
This skeleton contains only `IEEEtran.cls`, `IEEEtran.bst`, `IEEEtranS.bst`, IEEE bibliography abbreviation files, and a `main.tex` preamble. Copy to `paper/` to begin writing.
